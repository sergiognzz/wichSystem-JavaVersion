# wichSystem-JavaVersion
Herramienta para descubrir posible SO en base al TTL para CTF`s inspirado en el famoso wichSystem.py de S4vitar, es un homenaje a S4vitar con un script programado en Java.


USAGE:
***bash
java wichSystem.java <IPADDRESS>
***

EXAMPLE:
***bash
java wichSyste.java 192.168.1.143
***


OUTPUT:
--------------------------------------------
Fetching TTL for IP address: 192.168.1.143
Detected OS: Linux
TTL value: 64
--------------------------------------------
